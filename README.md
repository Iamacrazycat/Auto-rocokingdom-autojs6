# Auto-rocokingdom-autojs6

本项目是基于 **AutoJS6** 与 **OpenCV/OCR** 开发的《洛克王国》手机端全自动挂机助手。专为联机抓精灵场景下的 2P 玩家提供智能辅助。

> [!TIP]
> 💡 **灵感来源**：[yorusacri/Auto-rocokingdom](https://github.com/yorusacri/Auto-rocokingdom)  
> 📚 **AutoJS6 官方文档**：[docs.autojs6.com](https://docs.autojs6.com/#/)

---

## 📅 更新日志 (2026/04/19)
*   **🚀 全面转向 OCR 识别**：因为autojs的opencv不支持SIFT算法，图像特征算法（Canny/Template Matching）兼容性较差，改用mlkit OCR识别。
*   **🦾 响应式点击**：通过识别文字坐标并计算物理图标相对高度偏移，实现跨屏幕分辨率兼容。
*   **🛡️ 智能判定**：基于屏幕关键字库比对（命中 ≥2 个关键字即确认战斗），误触率大幅降低。

---

## ⚠️ 已知问题
*   **高饱和度场景识别失效**：在部分技能特效导致的高饱和度画面下，OCR 可能发生识别漂移或失效。由于此类场景样本量较小，目前正在持续优化预处理算法以进行修复。

---

## 🎯 核心功能

*   **⚡ 聚能模式**：循环自动执行聚能操作。
*   **🏃 逃跑模式**：识别并自动执行逃跑与确认逻辑。
*   **🧠 智能挂机**：
    *   **精准识别**：根据屏幕背景滤镜颜色（HSV 空间）自动切换逻辑。
    *   **动态抉择**：检测到紫色背景（聚能战斗）时自动聚能；检测到其他背景（意外遇敌）时自动逃跑。

## 🚀 快速开始 (AutoJS6 使用指南)

1.  **准备环境**：
    *   在 Android 手机上安装 [AutoJS6](https://github.com/SuperMonster003/AutoJs6/releases)。
    *   开启手机的 **无障碍服务** 与 **显示在其他应用上（悬浮窗）权限**。
2.  **导入项目**：
    *   将本项目源码文件夹整体拷贝至手机 `/sdcard/脚本/` 目录下。
3.  **运行脚本**：
    *   打开 AutoJS6 App，找到并点击 `main.js` 启动。
    *   脚本会请求 **截图权限**，请务必勾选“不再显示”并点击“允许”。
    *   根据控制台提示选择需要的模式（1-3）。
    *   *注：启动后控制台会隐藏以防干扰 OCR 识别。*

## 📱 设备与分辨率适配

由于采用了 **OCR 识别 + 比例偏移算法**，本项目不再依赖特定分辨率的模板。
*   **理论适配**：所有 Android 手机（16:9, 18:9, 21:9 等）。
*   **极简兜底**：唯一保留的模板匹配为 `escape_yes.png`（用于确认点击“是”）。若该按钮点击无效，脚本会有输出提示。
