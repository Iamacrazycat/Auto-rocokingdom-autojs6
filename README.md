# Auto-rocokingdom-autojs6

本项目是基于 **AutoJS6** 框架开发的《洛克王国》手机端全自动挂机助手。利用底层 Java OpenCV 技术（无需额外安装插件），实现了无视分辨率畸变、无视颜色蒙版的超强跨屏幕视觉识别方案。

## 🎯 核心功能

*   **智能挂机机制**：使用 OpenCV 判定战斗背景颜色，智能区分“打怪聚能”与“意外遇敌逃脱”场景。
*   **双管齐下视觉引擎**：内置极速并行特征矩阵引擎（灰度高斯模糊通道 + Canny 跨分辨率通道），全方位抵御手机半透明 UI 叠层、游戏自身动画特效呼吸以及设备抗锯齿导致的失效。
*   **极高可用性**：彻底剔除了不稳定的 SIFT/OCR 依赖，完全通过原生 OpenCV 轮廓匹配接管点击事件，稳如磐石。

## 📱 设备与分辨率说明

1.  **参考基准**：本仓库包含的默认 `templates` 特征图是基于长宽比约 `16:9` 的基准提取的。
2.  **动态无感适配**：代码已原生包含了 `uniqueScales` 多档无级缩放引擎。若你的手机分辨率是常见的带鱼屏或者全面屏（如 `2772x1280` 此类），只要画面整体变动不大，引擎会在后台自行缩放比对，**无需修改代码**，即开即用！

## 🔧 如果你的设备完全无法识别，该怎么做？

受限于个别手机极其特殊的显示比例拉伸或者魔改的 Android 分辨率。如果脚本启动后提示一直处于“等待”或“连续丢失”，你需要进行**一步极简单的换图工作**以永久适应你的设备：

1.  启动游戏并进入对应的界面（比如战斗画面）。
2.  直接使用手机自带的截屏功能截取全屏大图。
3.  在相册里对**带有文字与上方完整图标**的按钮部位进行框选裁剪（建议包含文字外的明显黑色边框或形状）。
4.  将裁剪好的新图片命名替换掉项目 `templates/` 目录里的同名文件。
    *   *进阶提示：如果在正常界面能跑，唯独“紫底”滤镜下跑不通，可以专门在这个形态下截取变体图片，并命名为 `特征名_purple.png` 扔进文件夹（如 `escape_btn_purple.png`）。代码具备多形态阵列寻图功能，会在找不到普通版时自动切换变体。*
