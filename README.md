# Auto-rocokingdom-autojs6

本项目是基于 **AutoJS6** 与 **OpenCV** 开发的《洛克王国》手机端全自动挂机助手。专为联机抓精灵场景下的 2P 玩家提供智能辅助。

> [!TIP]
> 💡 **灵感来源**：[yorusacri/Auto-rocokingdom](https://github.com/yorusacri/Auto-rocokingdom)  
> 📚 **AutoJS6 官方文档**：[docs.autojs6.com](https://docs.autojs6.com/#/)

---

## 🎯 核心功能

*   **⚡ 聚能模式**：循环自动执行聚能操作。
*   **🏃 逃跑模式**：识别并自动执行逃跑与确认逻辑。
*   **🧠 智能挂机**：
    *   **精准识别**：根据屏幕背景滤镜颜色（HSV 空间）自动切换逻辑。
    *   **动态抉择**：检测到紫色背景（聚能战斗）时自动聚能；检测到其他背景（意外遇敌）时自动逃跑。

## 🚀 快速开始 (AutoJS6 使用指南)

1.  **准备环境**：
    *   在 Android 手机上安装 [AutoJS6](https://github.com/SuperMonster003/AutoJs6/releases)。
    *   开启手机的 **无障碍服务** 与 **悬浮窗权限**。
2.  **导入项目**：
    *   推荐使用电脑端 **VSCode** 配合 AutoJS6 扩展进行同步。
    *   或将本项目源码文件夹整体拷贝至手机 `/sdcard/脚本/` 目录下。
3.  **运行脚本**：
    *   打开 AutoJS6 App，找到并点击 `main.js` 启动。
    *   脚本会请求 **截图权限**，请务必勾选“不再显示”并点击“允许”。
    *   根据控制台提示选择需要的模式（1-3）。

## 📱 设备与分辨率适配

1.  **多档位缩放引擎**：内置 `uniqueScales` 逻辑，支持 `16:9`、`18:9`、`21:9` 等多种比例的自动适配，理论上无需修改即可兼容大部分全面屏。
2.  **强制覆盖方案**：如果默认模板无法识别，请执行以下步骤：
    *   在手机上截取包含 **完整图标+文字** 的按钮图。
    *   替换 `templates/` 目录下的对应文件。
    *   **针对紫底滤镜**：若在紫底时失效，请截取紫底状态下的按钮并命名为 `xxx_purple.png`。脚本会自动寻找带后缀的变体。
